#Project Overview This project aims to train a machine learning model to predict patient risk levels and treatment priority based on age and disease severity.

The model analyzes structured healthcare data and calculates a final risk score used to classify patient priority

#نظرة عامة على المشروع

يهدف هذا المشروع إلى تدريب نموذج تعلم آلي للتنبؤ بمستوى المخاطر الصحية وأولوية العلاج للمرضى اعتمادًا على العمر وشدة المرض.

يقوم النموذج بتحليل بيانات صحية منظمة، ثم يحسب درجة الخطورة النهائية (Final Risk Score) والتي تُستخدم لتصنيف أولوية المريض وتحديد مستوى التدخل المطلوب.
